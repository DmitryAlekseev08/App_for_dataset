Структура проекта:
 * папка app содержит дистрибутив программы;
 * папка design содержит файлы с пользовательским интерфейсом программы (не код, а оформление в дизайнере);
 * папка picture содержит изображения, использованные в пользовательском интерфейсе;
 * файл Add_delete_window.py включает в себя код интерфейса окон для добавления\ удаления интентов, методы обработки сигналов при нажатии кнопок;
 * файл Change_window.py включает в себя код интерфейса окна для изменения названия интентов, методы обработки сигналов при нажатии кнопок;
 * файл Main_window.py включает в себя код интерфейса основного окна приложения;
 * файл main.py включает в себя код точки входа в программу, методы для обработки данных и передачи их между окнами.
